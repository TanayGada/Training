# DP problem usually asks to
1. Count all possible ways
2. Get min/max

# Steps to Follow in any DP Problem
1. Try to represent the problem in terms of index
2. Do all possible stuff on these index according to the problem statement
3. According to que
    a. Sum of all stuffs -> find all possible
    b. max of all stuffs -> find maximum
    c. min of all stuff -> find min
