- A pure virtual function in C++ is a virtual function that is declared by assigning 0 to it within a base class. This makes the base class abstract, meaning it cannot be instantiated directly and must be inherited by derived classes which provide implementations for the pure virtual function. Pure virtual functions enforce that derived classes implement specific behaviors, ensuring a consistent interface.

### Real-Life Example: Animal Communication System

Consider an animal communication system where different animals communicate in various ways. We can use a pure virtual function to enforce that all derived animal classes implement their own way of communicating.

#### Base Class: Animal

The base class `Animal` has a pure virtual function `communicate`, which forces all derived classes to implement their own version of `communicate`.

```cpp
#include <iostream>
#include <string>

// Base class
class Animal {
public:
    virtual void communicate() = 0; // Pure virtual function
};

```

#### Derived Class: Dog

The `Dog` class inherits from `Animal` and implements the `communicate` function.

```cpp
class Dog : public Animal {
public:
    void communicate() override {
        std::cout << "Dog barks: Woof! Woof!" << std::endl;
    }
};
```

#### Derived Class: Cat

The `Cat` class inherits from `Animal` and implements the `communicate` function.

```cpp
class Cat : public Animal {
public:
    void communicate() override {
        std::cout << "Cat meows: Meow! Meow!" << std::endl;
    }
};
```

#### Derived Class: Bird

The `Bird` class inherits from `Animal` and implements the `communicate` function.

```cpp
class Bird : public Animal {
public:
    void communicate() override {
        std::cout << "Bird sings: Tweet! Tweet!" << std::endl;
    }
};
```

### Example Usage

```cpp
int main() {
    Animal* animal;

    Dog dog;
    Cat cat;
    Bird bird;

    // Dog communicates
    animal = &dog;
    animal->communicate();

    // Cat communicates
    animal = &cat;
    animal->communicate();

    // Bird communicates
    animal = &bird;
    animal->communicate();

    return 0;
}
```

### Explanation

- **Animal Class**: The base class with a pure virtual function `communicate`, making `Animal` an abstract class. This ensures that any derived class must implement the `communicate` function.
- **Dog, Cat, Bird Classes**: Derived classes that provide their specific implementations of the `communicate` function.

### Output

```
Dog barks: Woof! Woof!
Cat meows: Meow! Meow!
Bird sings: Tweet! Tweet!
```

### Summary

- A pure virtual function is declared by assigning 0 to a virtual function within a base class.
- The base class becomes abstract and cannot be instantiated.
- Derived classes must implement the pure virtual function, ensuring a consistent interface.

Using the animal communication system as an example, the `Animal` class enforces that all derived classes (Dog, Cat, Bird) implement their own way of communicating, providing a clear and consistent interface for different types of animals.
