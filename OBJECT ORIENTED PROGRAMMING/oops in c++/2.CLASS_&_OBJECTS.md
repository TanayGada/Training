## C++ Object & Class
##### Definitions
- Class: A class is a blueprint or template for creating objects. It defines a set of attributes and methods that the created objects (instances of the class) will have.
- Object: An object is an instance of a class. It represents a specific entity that has the attributes and methods defined by the class.

##### Cricket Example: Player
- Let’s use a cricket player as an example to explain classes and objects.

###### Class: Player
- A cricket player can be represented as a class. The class defines the attributes and methods that a cricket player object will have.

###### Attributes (Data Members)
	•	name: The name of the player
	•	age: The age of the player
	•	role: The role of the player (e.g., batsman, bowler, all-rounder)
	•	runs_scored: The total runs scored by the player
	•	wickets_taken: The total wickets taken by the player

Methods (Member Functions)

	•	score_runs(runs): Method to add runs to the player’s total
	•	take_wicket(): Method to increment the player’s wicket count
	•	display_stats(): Method to display the player’s statistics

#### Example in C++
```cpp
#include <bits/stdc++.h>
using namespace std;

class Player {
private:
    string name;
    int age;
    string role;
    int runs_scored;
    int wickets_taken;

public:
    // Constructor
    Player(string playerName, int playerAge, string playerRole)
        : name(playerName), age(playerAge), role(playerRole), runs_scored(0), wickets_taken(0) {}

    // Method to add runs to the player's total
    void score_runs(int runs) {
        if (runs > 0) {
            runs_scored += runs;
            cout << name << " scored " << runs << " runs." << endl;
        }
    }

    // Method to increment the player's wicket count
    void take_wicket() {
        wickets_taken++;
        cout << name << " took a wicket." << endl;
    }

    // Method to display the player's statistics
    void display_stats() const {
        cout << "Player: " << name << endl;
        cout << "Age: " << age << endl;
        cout << "Role: " << role << endl;
        cout << "Runs Scored: " << runs_scored << endl;
        cout << "Wickets Taken: " << wickets_taken << endl;
    }
};

int main() {
    // Creating a cricket player object
    Player player1("Virat Kohli", 35, "Batsman");

    // Performing operations on the player object
    player1.score_runs(100);
    player1.take_wicket();  // Not typically done by a batsman, but for example's sake
    player1.display_stats();

    return 0;
}
```
