### Understanding `static` Keyword with Real-Life Example

The `static` keyword in C++ can be used in various contexts: static variables in functions, static member variables, and static member functions in classes. It changes the behavior and scope of the variables and functions in different ways.

### Static Variable in a Function

A static variable in a function retains its value between function calls.

#### Real-Life Example: Counter in a Ticket Machine

Imagine a ticket machine that keeps track of the total number of tickets issued, even after the machine is reset.

```cpp
#include <iostream>

void issueTicket() {
    static int ticketCount = 0; // Static variable retains its value between function calls
    ticketCount++;
    cout << "Ticket number: " << ticketCount << endl;
}

int main() {
    issueTicket(); // Output: Ticket number: 1
    issueTicket(); // Output: Ticket number: 2
    issueTicket(); // Output: Ticket number: 3
    return 0;
}
```

### Static Member Variable in a Class

A static member variable is shared among all instances of the class. It is not tied to any particular object but to the class itself.

#### Real-Life Example: Shared Bank Interest Rate

Imagine a bank where all accounts share the same interest rate. This interest rate is common for all account holders.

```cpp
#include <iostream>

class BankAccount {
public:
    static double interestRate; // Static member variable

    BankAccount() {}

    static void setInterestRate(double rate) {
        interestRate = rate;
    }

    void displayInterestRate() const {
        cout << "Current interest rate: " << interestRate << "%" << endl;
    }
};

// Definition of the static member variable
double BankAccount::interestRate = 0.0;

int main() {
    BankAccount::setInterestRate(3.5); // Setting interest rate for all accounts

    BankAccount account1;
    account1.displayInterestRate(); // Output: Current interest rate: 3.5%

    BankAccount account2;
    account2.displayInterestRate(); // Output: Current interest rate: 3.5%

    return 0;
}
```

### Static Member Function in a Class

A static member function can be called without creating an instance of the class. It can access static member variables but cannot access non-static member variables.

#### Real-Life Example: Utility Functions for a Calculator

Imagine a calculator class with static utility functions for basic arithmetic operations.

```cpp
#include <iostream>

class Calculator {
public:
    static int add(int a, int b) {
        return a + b;
    }

    static int subtract(int a, int b) {
        return a - b;
    }
};

int main() {
    // Using static member functions without creating an instance of the class
    cout << "Addition: " << Calculator::add(10, 5) << endl; // Output: Addition: 15
    cout << "Subtraction: " << Calculator::subtract(10, 5) << endl; // Output: Subtraction: 5

    return 0;
}
```

### Summary

- **Static Variable in a Function**: Retains its value between function calls.
  - **Real-Life Example**: Counter in a ticket machine.

- **Static Member Variable in a Class**: Shared among all instances of the class.
  - **Real-Life Example**: Shared bank interest rate for all account holders.

- **Static Member Function in a Class**: Can be called without an instance of the class and can only access static members.
  - **Real-Life Example**: Utility functions for a calculator.

### Detailed Breakdown

#### Static Variable in a Function

```cpp
void issueTicket() {
    static int ticketCount = 0; // Retains value between calls
    ticketCount++;
    cout << "Ticket number: " << ticketCount << endl;
}
```

- **Scope**: Local to the function.
- **Lifetime**: Exists for the entire duration of the program.
- **Behavior**: Retains its value across multiple calls to the function.

#### Static Member Variable in a Class

```cpp
class BankAccount {
public:
    static double interestRate; // Shared among all instances

    static void setInterestRate(double rate) {
        interestRate = rate;
    }

    void displayInterestRate() const {
        cout << "Current interest rate: " << interestRate << "%" << endl;
    }
};

// Definition outside the class
double BankAccount::interestRate = 0.0;
```

- **Scope**: Shared among all instances of the class.
- **Lifetime**: Exists for the entire duration of the program.
- **Behavior**: Common to all objects of the class.

#### Static Member Function in a Class

```cpp
class Calculator {
public:
    static int add(int a, int b) {
        return a + b;
    }

    static int subtract(int a, int b) {
        return a - b;
    }
};
```

- **Scope**: Can be called without an instance of the class.
- **Lifetime**: Exists for the entire duration of the program.
- **Behavior**: Can only access static members of the class.

By understanding the `static` keyword and its various uses, you can effectively manage data and functions that need to be shared or retain state across different parts of your program.
